EssentialsX is the most widely used Minecraft server plugin suite, providing everything from basic utility commands to a full economy system, chat formatting, Discord bridging, and spawn management. It runs on CraftBukkit, Spigot, and Paper for Minecraft 1.8.8 through the latest versions.

## Get started

<Steps>
  <Step title="Download EssentialsX">
    Grab the latest release from [essentialsx.net/downloads.html](https://essentialsx.net/downloads.html). Download the core `EssentialsX.jar` and any module JARs you need.
  </Step>
  <Step title="Install on your server">
    Place all downloaded JARs into your server's `plugins/` folder. For economy features, also install [Vault](https://dev.bukkit.org/projects/vault) and a permissions plugin such as [LuckPerms](https://luckperms.net).
  </Step>
  <Step title="Start your server">
    Start (or restart) your server. EssentialsX will generate its configuration files in `plugins/Essentials/`. Stop the server to edit `config.yml` before going live.
  </Step>
  <Step title="Configure and assign permissions">
    Edit `plugins/Essentials/config.yml` for global settings, then use your permissions plugin to grant `essentials.*` to ops or specific nodes to player groups. Run `/ess reload` to apply config changes without a restart.
  </Step>
</Steps>
