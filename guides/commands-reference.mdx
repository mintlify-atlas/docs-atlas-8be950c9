---
title: "Commands Reference"
description: "Complete reference for every command provided by the core EssentialsX plugin, organized by category."
---

This page lists every command provided by the core **EssentialsX** plugin. Commands from addon modules (EssentialsX Chat, EssentialsX Spawn, EssentialsX Discord, etc.) are documented separately.

All commands require the corresponding `essentials.<command>` permission. Operators receive `essentials.*` by default, which grants all permissions. See [Permissions](/guides/permissions) for details on assigning individual nodes.
